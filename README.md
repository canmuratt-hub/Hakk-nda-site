# 🌌Kişisel Kartvizit

## 📝 Proje Hakkında
Bu proje, dijital kimliğimi ve iletişim kanallarımı tek bir çatı altında toplayan, modern web tasarım trendlerine uygun olarak geliştirilmiş kişisel bir web sayfasıdır. 

## 💻 Kullanılan Teknolojiler
* **HTML5:** Sayfa iskeleti ve semantik yapı.
* **CSS3:** Görsel şekillendirme, Flexbox mimarisi, animasyonlar ve pseudo-element (`::before`) kullanımı.
